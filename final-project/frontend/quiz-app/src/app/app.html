Get Started

To get started, you have to decide if your a QC or a QT. A QC is a Quiz Creator, they create quizzes for others to take. A QT is a Quiz Taker, they take quizzes created by QCs. If you want to create quizzes, you have to register as a QC. If you want to take quizzes, you have to register as a QT.
